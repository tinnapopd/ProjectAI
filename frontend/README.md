## Installation

```bash
npm install
```

## Running the App

```bash
npm start
```

The app will open at [http://localhost:3000](http://localhost:3000)

## Backend Connection

Make sure the backend API is running at `http://localhost:8000`

## Usage

1. Enter your company name and competitor
2. Set your business goal (e.g., "Achieve 10% revenue growth")
3. Click "Start Game" to create a session
4. Click "Generate Strategic Moves" to get AI-powered moves
5. Click on any move to visualize its decision tree
